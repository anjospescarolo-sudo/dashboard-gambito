# Dashboard Gambito

Painel de desempenho e simulação de bônus trimestral da **Equipe Gambito** (assessoria de investimentos).

- **App:** painel estático (um único HTML) publicado via GitHub Pages.
- **Acesso:** https://anjospescarolo-sudo.github.io/dashboard-gambito/ (protegido por senha).
- **Dados:** apenas indicadores de desempenho — sem dados pessoais de clientes.

O frontend lê e grava os indicadores em uma planilha Google através de um backend Google Apps Script (mantido fora deste repositório).
